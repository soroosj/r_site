A collection of data visualizations I have built for fun using the ggplot2 package in the [R] (https://www.r-project.org/about.html)programming language. All visualizations are the result of the code without any post-processing.

TidyTuesday

#TidyTuesday is a weekly data project aimed at the R ecosystem and tidyverse package bundle to source, wrangle, explore and visualize the data set provided by the R4DS Online Learning Community.
